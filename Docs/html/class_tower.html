void Tower.SelectTower ( )

Selects the tower.

Selects this tower and renders its range if it's not already selected. Unselects all towers and selects this tower if another tower was already selected.

void Tower.ShootCritter ( Critter  target)

Shoots the critter.

Creates a bullet instance and sets its relevant attributes to tower attributes and makes the parameter critter the target of the bullet only if the tower can shoot; otherwise, does nothing.
